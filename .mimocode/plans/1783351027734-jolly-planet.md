# Plan: Add WYSIWYG Editor (django-ckeditor) for Blog Content

## Goal
Replace the plain `TextField` for blog post content with a rich text editor in Django admin so the user can write styled articles without writing raw HTML.

## Changes

### 1. Install django-ckeditor
- `pip install django-ckeditor`

### 2. Update `config/settings.py`
- Add `'ckeditor'` to `INSTALLED_APPS`

### 3. Update `blog/models.py`
- Change `content = models.TextField()` to `content = CKEditor5Field()` (or `CKEditorWidget`-compatible field)
- Import from `ckeditor5.fields`

### 4. Update `blog/admin.py`
- The `content` field will automatically use the CKEditor widget once the model field is changed

### 5. Create CKEditor config (optional)
- Add `CKEDITOR_5_CONFIGS` in settings for custom toolbar (headings, bold, italic, code blocks, images, blockquotes, lists)

## Files to modify
- `config/settings.py`
- `blog/models.py`
- `blog/admin.py`

## Verification
- Run `python manage.py makemigrations blog && python manage.py migrate`
- Run `python manage.py check`
- Create/edit a post in admin — confirm the content field shows a rich text editor toolbar
- Write a post with headings, bold, code blocks, blockquote — confirm it renders correctly on the blog detail page
